Installation

Windows

Pour installer TA3D sous Windows XP lancez simplement ta3d-0-4-2.exe et décompresser son contenu dans un nouveau dossier (par exemple C:/ta3d/) puis lancez install.bat avec le CD-ROM de TA dans votre lecteur cdrom (Cela copiera totala1.hpi et totala2.hpi dans le dossier de TA3D).

Vous pouvez aussi copier totala1.hpi et totala2.hpi à la main.

Lancez TA3D.exe pour lancer TA3D. (impressionant, hein?)

Note: En cas de plantage essayez de réduire la qualité de l'eau, celle-ci utilise les shaders 2.0 à partir de "normal".

Linux

Comment compiler TA3D soi-même

Pour compiler TA3D sous Linux vous aurez besoin des librairies Allegro, AllegroGL, FMODex 4 et zlib installées avec leurs fichiers de développement ainsi que le paquet source de TA3D.
Il vous faut aussi le compilateur GCC avec le support du C++(g++).

Pour compiler TA3D tapez dans une console:
# ./configure
# make

Pour installer les fichiers ressources lancez install-sh avec le CD-ROM de TA dans votre lecteur cdrom (Cela copiera totala1.hpi et totala2.hpi dans le dossier de ta3d).
Pour lancer TA3D:
# ./ta3d

Comment installer un paquet binaire

Des paquets binaires sont maintenant disponibles pour Linux, pour systèmes 32 et 64 bits. Vous ne devriez pas avoir besoin d'autres paquets que ceux disponibles sur notre site web (ils contiennent les versions requises d'Allegro et AllegroGL ainsi que la librairie FMODex). Décompactez simplement l'archive qui correspond le mieux à votre système et lancer le script install-sh pour installer les ressources requises depuis les cds de Total Annihilation.

NB: si ces paquets ne fonctionnent pas sur votre système (erreurs à propos de quelque chose qui manque ou qui n'est pas compatible) vous pouvez essayer de compiler TA3D vous-même depuis les sources qui ne sont pas incluses dans les paquets binaires.
Pour lancer TA3D:
# ./ta3d
