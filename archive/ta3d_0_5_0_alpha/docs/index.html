Main page


TA3D is an attempt to bring TA into the world of true 3D, it is being developed by those who love to still play TA, and is by no means being backed by any large corporation. It is being developed in our spare time by a bunch of hobbiest programers with a love for the game.


That means TA3D uses resources from original Total Annihilation:

Different ways to run TA3D

- to run in light configuration (able to play scirmish games with basic TA set of units) you`ll need:
totala1.hpi for resources and totala2.hpi for maps.
totala2.hpi can be replaced with any *.ufo,*.hpi,*.gp3, *.ccx map or mappack.
(this will allow playing with TA set of units)

- to run in normal configuration (able to play scirmish games with full set of TA units) you`ll need:
totala1.hpi for resources, totala2.hpi for maps, ccdata.ccx, rev31.gp3 from TA: Core Contingency 3.1c.
(this will allow playing with TA: Core Contingency set of units)
totala2.hpi can be replaced with any *.ufo,*.hpi,*.gp3, *.ccx map or mappack
several map/mapppacks can be in /ta3d/ folder at the same time

- to run in full configuration (able to play scirmish games, TA campaigns!!! with full set of TA units) you`ll need:
totala1.hpi for resources, totala2.hpi for maps, ccdata.ccx, rev31.gp3, totala4.hpi, ccdata.ccx, ccmaps.ccx, ccmiss.ccx from TA: Core Contingency.
(this will allow to play compaigns from TA and TA: Core Contingency)
totala2.hpi can be replaced with any *.ufo,*.hpi,*.gp3, *.ccx map or mappack
several map/mapppacks can be in /ta3d/ folder at the same time

- to run in custom configuration you`ll need:
totala1.hpi for resources and any map/mappack.
Feel free to experiment with different mods (place mod resource files in /ta3d/mods/ folder).
Note:See more about using mods in Using Mods section of this help.

TA3D compatibility with mods hasn't been tested fully yet (it is only v.0.4.2), so the game might crash sometimes or some units may don`t behave correctly, but the main goal is full compatibility with Total Annihilation and most of it`s mods.

Some mods may need a small correction before using in TA3D, that`s because of many "hack" stuff used for overriding original TA engine in some cases. TA3D have no need in such things, and modders will be able to port their mods in a couple of minutes.

TA3D runs on following platforms:
Windows 32bits
Windows 64bits
Linux 32bits (x86)
Linux 64bits (x86_64)
